My name is Erick Otenyo, and while I do a lot of different things, the majority of the time I’m a GIS developer.

I’m currently studying Geospatial Science at Jomo Kenyatta University- Kenya, where I’m in my final year,and soon hopefully I will be joining the GIS developing world..

I have been writing code for the past 2 years now, most of it being on building Web GIS applications, with maps as the main interface . You can see more of this on the Projects page

I mostly code in python with Django, to build web GIS solutions. I also code stuff not related to GIS, and usually find a way to spatially enable them, because as it is known 'everything happens somewhere'..

Apart from that, I like giving talks on what I do, dancing to African Music, and spending time with friends, while not letting them use Windows because windows sucks..

I use Linux all the time.
